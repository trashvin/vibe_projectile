# Vibe Projectile

This is a Pyglet-based simulation app scaffolded from `ai/main` and `ai/prompts`.

## Run setup

1. `bash setup/bootstrap.sh`
2. `bash setup/install.sh`

## Run the app

- `bash run-main.sh`

## Run tests

- `pytest -q`

## Project structure

- `src/` app content
- `setup/` bootstrap/install scripts
- `tests/` unit tests
- `requirements.txt` dependencies
